# RaspberryPiPicoW_host_monitor

Sencillo monitor de red con Raspberry Pi Pico W
Es como una "especie de PING" ya que no encontr´e en MicroPython la existencia del comando "ping"
Los hosts deben tener al menos un puerto abierto!

En caso de falla de alguno de ellos, el led onboard parpadea rápidamente.

Tiene una sencilla web embebida que muestra un breve historial, y en caso de falla, indica que host no es accesible.
